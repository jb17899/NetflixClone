# 🎬 Netflix Clone

A full-stack Netflix-inspired streaming platform that allows users to browse, search, and save movies. Built with React on the frontend, Node.js/Express on the backend, and MongoDB for the database.

---

## 🌐 Tech Stack

- **Frontend:** React, Tailwind CSS, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose ODM)
- **Authentication:** JWT
- **Movie Data Source:** TMDB API (The Movie Database)

---

## ✨ Features

- 🔐 User authentication and session management
- 🧾 Browse movies by category (Trending, Originals, Top Rated, etc.)
- 🔍 Search movies by title
- 🎥 Watch trailers via YouTube
- ❤️ Add/remove movies from personalized "My List"
- 📱 Fully responsive design
